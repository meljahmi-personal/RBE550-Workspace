RBE 550 Grid Planning Benchmark

A ROS 2 package for benchmarking classical grid-based path planning algorithms with RViz visualization.
Features

    Algorithms: BFS, Dijkstra, Greedy Best-First, A, Weighted A

    Grid Types: Random grids or ASCII map files

    Connectivity: 4-connected or 8-connected moves

    Visualization: RViz2 integration with occupancy grids, paths, and markers

    Metrics: Path length, nodes expanded, runtime

Quick Start
1. Build the Workspace
bash

./scripts/dev.sh

2. Launch with RViz Visualization
bash

./scripts/run_rviz.sh

3. Run Algorithm Benchmarks (CLI)
bash

source install/setup.bash

# Basic A* example
ros2 run rbe550_grid_bench bench --algo astar --grid 64 --fill 0.2 --show

# All algorithms comparison
ros2 run rbe550_grid_bench bench --algo bfs --grid 32 --fill 0.2 --seed 42 --show
ros2 run rbe550_grid_bench bench --algo dijkstra --grid 32 --fill 0.2 --seed 42 --show
ros2 run rbe550_grid_bench bench --algo greedy --grid 32 --fill 0.2 --seed 42 --show
ros2 run rbe550_grid_bench bench --algo astar --grid 32 --fill 0.2 --seed 42 --show

RViz Visualization

The package includes a ROS 2 node that publishes to RViz:

    OccupancyGrid: /grid - Shows obstacles and free space

    Path: /path - Shows the planned path (green line)

    Markers: /markers - Shows start (green sphere) and goal (red sphere)

Launch with Custom Parameters
bash

# Change algorithm via launch parameters
./scripts/run_rviz.sh algo:=bfs
./scripts/run_rviz.sh algo:=dijkstra
./scripts/run_rviz.sh algo:=astar moves:=4
./scripts/run_rviz.sh algo:=weighted_astar weight:=2.0

Performance Testing

For batch testing without visualization:
bash

# Remove --show for faster execution
ros2 run rbe550_grid_bench bench --algo astar --grid 64 --fill 0.2
ros2 run rbe550_grid_bench bench --algo bfs --grid 64 --fill 0.2

# With specific random seed for reproducibility
ros2 run rbe550_grid_bench bench --algo astar --grid 64 --fill 0.2 --seed 12345

CLI Options Reference
Option	Description	Values
--algo	Planning algorithm	bfs, dijkstra, greedy, astar, weighted_astar
--moves	Movement connectivity	4, 8
--weight	Heuristic weight (A* only)	float > 0
--grid	Grid size N for N×N	int
--fill	Obstacle density	0.0 - 1.0
--seed	Random seed	int
--map	ASCII map file path	string
--start-goal	Start and goal positions	"r1,c1:r2,c2"
--show	Enable visualization	flag
--gif	Save animation as GIF	file path
Output Metrics

Each run provides:

    Success: Whether a path was found

    Path Length: Number of steps in solution

    Runtime: Planning time in milliseconds

    Nodes Expanded: Search space exploration

    Grid Info: Dimensions and obstacle count

Troubleshooting

RViz shows "No map received":

    Ensure planner node is running: ros2 node list

    Check topic publishing: ros2 topic echo /grid

    Verify RViz display configuration

Algorithm not found:

    Check algorithm spelling: bfs, dijkstra, greedy, astar, weighted_astar

    Rebuild package: ./scripts/dev.sh

No path found:

    Reduce obstacle density with --fill

    Try different start-goal positions

    Use 8-connected moves with --moves 8
